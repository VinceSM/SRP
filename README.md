# 🧩 Aplicación Demostración de SRP (Single Responsibility Principle)

Aplicación de escritorio en **C# con Windows Forms** diseñada específicamente para demostrar el **Principio de Responsabilidad Única (SRP)** de SOLID. Cada clase tiene **una y solo una razón para cambiar**, aplicando arquitectura MVC limpia y separación extrema de responsabilidades.
